# Weather-App
Простой и удобный веб-сайт для просмотра текущей погоды по названию города. 
Использует OpenWeatherMap API для получения данных о температуре, влажности, скорости ветра и погодных иконках.

# Figma макет

# Функционал 
Поиск погоды по городу  
Отображение: 
* Название города
* Температура
* Иконки погодных условий
* Влажности воздуха
* Скорость ветра

# Технологии

* CSS
* JavaScript 
* OpenWeatherMap API
